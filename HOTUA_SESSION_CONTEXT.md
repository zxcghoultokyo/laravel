## Технічний стек

- **Laravel 12 + PHP 8.4** — бекенд
- **Filament 3** — адмін-панель
- **Nutgram (nutgram/nutgram)** — Telegram Bot SDK
- **OpenAI GPT-4o-mini** — генерація текстів + скоринг ділів (1-10)
- **DALL-E 3** — генерація зображень для абстрактних постів (розіграші, лайфхаки)
- **Intervention Image v3** — PHP композиція зображень (фото товару + overlay з ціною, знижкою, логотипом, @zlovyv)
- **SalesDoubler API** — `https://my.salesdoubler.pro/api/v1/` (Bearer token auth)
- **Redis + MySQL + Laravel Queue** — асинхронна обробка
- **Multi-provider affiliate архітектура** — інтерфейс + імплементації (SalesDoubler → Admitad → прямі партнерства)

## Дослідження SalesDoubler API

### Знайдені ендпоінти
- `https://my.salesdoubler.pro/api/v1/offers` — список оферів (401 без токена)
- `https://my.salesdoubler.pro/api/v1/conversions` — конверсії (401 без токена)

### Автентифікація
- **Метод:** `Authorization: Bearer {token}`
- **Підтвердження:** Bearer повертає "Доступ запрещен!" (інший error), інші методи — "У доступі відмовлено!"
- **Токен отримується:** в кабінеті my.salesdoubler.pro після реєстрації

### Формат діплінку
```
https://my.salesdoubler.pro/go/{offer_id}/?ulp={encoded_url}&sub_id=hotua&sub_id_2={post_id}
```

### Product Feeds
- SalesDoubler надає Product Feed (XML/CSV) по рекламодавцям
- Це ОСНОВНЕ джерело даних про товари зі знижками
- URL фідів доступні в кабінеті після підключення до оферу

## Пайплайн створення посту (5 кроків)

### 1. 📥 Збір (Fetch)
```
Cron: deals:fetch кожні 30 хв
```
- Завантажити Product Feed (XML/CSV) від SalesDoubler
- Парсити: назва, ціна, стара ціна, URL, зображення, категорія
- Генерувати affiliate URL через deeplink
- Зберегти в `deals` таблицю зі статусом `new`
- Дедуплікація по URL товару

### 2. 🏆 Оцінка (Score)
```
Автоматично після fetch
```
- **Формула скорингу (1-10):**
  - Розмір знижки (>50% = макс бали)
  - Популярність магазину
  - Категорія (електроніка > одяг > аксесуари)
  - Терміновість (обмежений час)
- GPT-4o-mini для семантичного скорингу (чи реально цікава знижка?)
- Поріг публікації: score ≥ 6
- Автовідхилення: score < 4

### 3. ✍️ Генерація (Generate)
```
Cron: schedule:generate щодня о 07:00
```
- Вибрати топ-N ділів зі score ≥ 6
- Розподілити по рубриках (9 PostType):
  1. `deal_of_day` — Діл дня
  2. `flash_sale` — Блискавична знижка
  3. `comparison` — Порівняння цін
  4. `collection` — Підбірка
  5. `battle` — Батл товарів
  6. `fake_or_real` — Фейк чи реальна знижка?
  7. `anomaly_alert` — Аномальна ціна
  8. `lifehack` — Лайфхак
  9. `raffle` — Розіграш
- GPT генерує текст по шаблону рубрики
- Intervention Image створює картинку (фото товару + overlay)
- Зберегти в `posts` зі статусом `scheduled`, запланувати час ±25 хв

### 4. 📤 Публікація (Publish)
```
Cron: posts:publish кожну хвилину
```
- Перевірити чи є пости з `scheduled` + час ≤ now
- Відправити в @zlovyv через Nutgram
- Оновити статус → `published`
- Зберегти `telegram_message_id`

### 5. 📊 Трекінг (Track)
```
Cron: stats:sync кожні 2 години
```
- Запит `GET /api/v1/conversions` з Bearer токеном
- Матчити конверсії по `sub_id_2` (post_id)
- Оновити статистику: кліки, конверсії, дохід
- Dashboard в Filament
