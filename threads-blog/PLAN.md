# Threads блог - план

## Концепція

Анонімний блог про бізнес-ідеї, продукти які роблю, і досвід вайбкодінгу.
Без прив'язки до реального імені. Цільова аудиторія - люди які теж щось будують, шукають ідеї, або просто цікавляться як це працює зсередини.

Можливі бонуси:
- знайти аудиторію кому це цікаво
- знайти клієнтів на свої продукти
- знайти вкладників або партнерів

## Профіль

**Біо (коротка):**

> роблю продукти. аі, екомерс, автоматизація. пишу про ідеї і що з ними далі відбувається. іноді працює іноді ні

**Альтернативні варіанти біо:**

> будую штуки які заробляють. або мали б заробляти. пишу як є

> кілька продуктів в роботі. аі екомерс автоматизація. ділюсь процесом без прикрас

> роблю продукти в соло. аі чатботи, автоматизація, екомерс. пишу що реально відбувається

## Продукти і проекти (контент-база)

### 1. Aintento - аі чатбот для екомерсу
- працює на GPT, зараз під Horoshop, далі інші платформи
- є бета тестери, наближається реліз і платні пакети
- головна фішка - ентічмент товарів який працює для будь-якої ніші
- бот дає точні поради навіть на розмиті семантичні запити
- проблеми: соло розробка, код зростає, незрозуміло звідки брати бюджет на маркетинг
- сайт: https://aintento.laravel.cloud/

### 2. Zlovyv - тг канал зі знижками
- повністю автоматизований
- підключений до афіліейт платформ, сам шукає знижки і промокоди
- рубрикатор, графік, автопостінг
- автогенерація текстів і зображень
- в планах тг бот-додаток (міні апп) з гейміфікацією:
  - прокачка персонажа
  - нарощування знижок
  - кращі промокоди за активність
  - розіграші, колесо фортуни
- фронт робить друг, бек - я
- проблема: немає навіть 10к грн щоб протестити реакцію на канал
- канал: https://t.me/zlovyv

### 3. Вайбкодінг як робота
- працюю над реальним екомерс магазином в києві
- магазин працює, має клієнтів, комерційно успішний
- я вайбкоджу і отримую зп мідла погодинно
- 10-20 годин на місяць але прибавка норм
- можливо хейт, можливо цікавий кейс для інших

### 4. Contrroz - інтеграція Horoshop з Rozetka
- повна інтеграція ~1000 товарів між інтернет-магазином і маркетплейсом
- Laravel сервіс, деплой на Laravel Cloud
- 146 комітів, з них 80 (55%) - фікси і дебаг через недокументовані API flow
- автозаповнення характеристик Розетки з 6 джерел + fuzzy matching
- hash-based change detection щоб не спамити модерацію
- 20+ діагностичних ендпоінтів бо API мовчки ігнорує помилки
- детективна історія: пароль в base64, + в URL фото, double-encoded JSON
- зроблено як інтегратор на фрілансі
- сайт: https://contrroz.laravel.cloud/

### 5. Ідея: апка для вайбкодерів
- максимально просте налаштування середовища
- вибір мови, фреймворку, агентів
- готові спейси під конкретні стеки
- зручна обгортка VСCode для людей без досвіду
- підписочна модель
- рейтинг проектів, шерінг конфігів
- можливо маркетплейс агентів і промптів

## Стиль

- ніяких емодзі
- тільки тире "-" для списків
- рвана структура
- різна довжина речень
- як записка собі або повідомлення другу
- без канцеляризмів, без "шановні", без "розглянемо"
- іноді незакінчені думки - нормально
- можна суперечити собі
- якщо щось не працює - писати що не працює
