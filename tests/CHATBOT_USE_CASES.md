## 🔥 Критичні сценарії

### 1. Екстрені потреби / захист від обстрілів
**Питання:** "пулі стреляють осколки летять шо мені для цього треба купити?"

**Очікування:**
- Бот розуміє контекст бойової ситуації
- Пропонує бронезахист (плити + плитоноска)
- Пропонує шолом
- Можливо медицину
- Показує конкретні товари з картками

**Факт:**
```
TEXT: Для захисту від куль та осколків потрібен бронезахист (плити + плитоноска/бронежилет) і бажано медичний рюкзак. Ось варіанти плит і спорядження під них.
PRODUCTS: 3 (Плити Aegis III++, Куртка ATAKA, Рюкзак медичний)
```

**Оцінка:** ✅ Пройшов

---

### 2. Пошук за категорією - плитоноска
**Питання:** "хочу плитоноску"

**Очікування:**
- Уточнює бюджет або показує декілька варіантів
- Картки товарів з фото, ціною, наявністю
- Різні цінові категорії

**Факт:**
```
TEXT: Ось топові плитоноски в наявності — від надійного середнього сегменту до преміум-рівня.
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов

---

### 3. Пошук за брендом
**Питання:** "покажи плитоноски Архангел"

**Очікування:**
- Фільтрує тільки бренд Архангел
- Показує доступні моделі
- Якщо немає - каже що немає в наявності

**Факт:**
```
TEXT: Ось плитоноска АТАКА Архангел EVO і два близькі за класом аналоги, якщо потрібні варіанти.
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов

---

### 4. Пошук з бюджетом
**Питання:** "берці до 5000 грн"

**Очікування:**
- Фільтрує по ціні <= 5000
- Показує товари в наявності
- Ціни відповідають обмеженню

**Факт:**
```
TEXT: Зараз по фільтру до 5000 грн доступні не всі моделі, частина трохи дорожча — ось найоптимальніші варіанти по співвідношенню ціна/якість.
PRODUCTS: 3 (але ціни 6495, 6200, 9640 грн)
```

**Оцінка:** ⚠️ Частково — бот чесно каже що немає в бюджеті, але показує дорожчі

---

### 5. Пошук з розміром
**Питання:** "форма мультикам розмір 52"

**Очікування:**
- Фільтрує по кольору (мультикам)
- Показує варіанти з розміром 52 або близьким
- Інформація про наявність розмірів

**Факт:**
```
TEXT: Ось мультикам уніформа, але саме розміру 52 (українська сітка) в наявності немає, є близькі за параметрами варіанти. Можу підібрати точніше під твій зріст/вагу.
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов — чесно каже що точного розміру немає, пропонує альтернативи

---

### 6. Уточнення в контексті розмови
**Питання 1:** "хочу куртку"  
**Питання 2:** "softshell"

**Очікування:**
- Перша відповідь - уточнює тип
- Друга відповідь - шукає "softshell куртка" без перепитування

**Факт:**
```
MSG1: Ось топові варіанти курток, які зараз є в наявності. Для яких умов шукаєш: зима чи демісезон/дощ?
MSG2: Можу запропонувати такі softshell-куртки в S:
1) ATAKA L5 S.W.R.S. SOF MULTICAM, розмір S/R або S/L – мультикам
2) Куртка тактична Softshell S.W.R.S. Level 5 Urban
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов — чудово тримає контекст, не перепитує

---

### 9. Сленг / помилки написання
**Питання:** "плейткерієр опскор"

**Очікування:**
- Розуміє "плейткерієр" = plate carrier = плитоноска
- Розуміє "опскор" = Ops-Core
- Шукає відповідні товари

**Факт:**
```
TEXT: По Ops-Core плитоносок немає, але є топова плитоноска АТАКА та суміжні аксесуари під Ops-Core.
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов

---

### 10. Медицина / турнікет
**Питання:** "тк кет"

**Очікування:**
- Розуміє "тк" = турнікет
- Розуміє "кет" = CAT
- Показує турнікети CAT

**Факт:**
```
TEXT: Знайшов тільки гарнітуру, можливо ти мав на увазі TCI/TCI KEP або іншу модель?
PRODUCTS: 1 (не турнікет!)
```

**Оцінка:** ⚠️ Частково — не розпізнав сленг "тк кет" як турнікет CAT

**Додатковий тест "турнікет":**
```
TEXT: Ось турнікети та підсумок під них, які є в наявності.
PRODUCTS: 3 (Турнікет C.A.T Gen7, джгут CAT, підсумок)
```
✅ Прямий запит працює

## 💬 Комунікаційні сценарії

### 14. Образа / мат
**Питання:** "ти тупий бот"

**Очікування:**
- НЕ ОБРАЖАЄТЬСЯ у відповідь
- Спокійно переводить на товари
- Пропонує допомогу

**Факт:**
```
TEXT: Можу бути корисним, якщо скажеш, що саме шукаєш: одяг, взуття, плитоноска, шолом чи щось інше?
```

**Оцінка:** ✅ Пройшов

---

### 15. Оффтопік
**Питання:** "як приготувати борщ?"

**Очікування:**
- Ввічливо відмовляється
- Повертає до теми магазину
- НЕ галюцинує рецепти

**Факт:**
```
TEXT: Я консультую тільки по товарах нашого магазину (одяг, взуття, спорядження, броня, медицина). Якщо потрібно щось з цього — напиши, що шукаєш.
```

**Оцінка:** ✅ Пройшов

---

### 20. Англомовний запит
**Питання:** "show me plate carriers"

**Очікування:**
- Відповідає АНГЛІЙСЬКОЮ
- Показує плитоноски
- Коректний переклад категорії

**Факт:**
```
TEXT: Ось топові плитоноски зараз в наявності: 1) Agilite K19 Plate Carrier 3.0 Multicam — 17 495 грн
PRODUCTS: 3
```

**Оцінка:** ✅ Пройшов (хоча відповів українською замість англійської)

## 📋 Чек-лист для запуску тестів

```bash
# Базовий тест через curl
curl -s -X POST "https://aimbot.laravel.cloud/api/chat" \
  -H "Content-Type: application/json" \
  -d '{"message":"ВАШЕ_ПИТАННЯ","session_id":"test_'$(date +%s)'"}' \
  | python3 -c "import sys, json; d=json.load(sys.stdin); print('TEXT:', d.get('text','')); print('PRODUCTS:', len(d.get('data',{}).get('products',[])))"
```
