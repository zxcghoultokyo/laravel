## 🔧 РЕКОМЕНДАЦІЇ ДО ВИПРАВЛЕННЯ

### Пріоритет 1 (терміново):
1. **Обробка chitchat/подяки** - "дякую" повинно повертати "Будь ласка!"
2. **FAQ-питання** - "Як замовити?" повинно давати інфо, не товари

### Пріоритет 2 (важливо):
3. **Персоналізація відповідей** - замість "Ось що я знайшов" писати контекстні відповіді
4. **Порівняння товарів** - реалізувати порівняльну таблицю
5. **Новинки** - сортування по updated_at для "Що нового?"

### Пріоритет 3 (бажано):
6. Підтвердження фільтрів в follow-up
7. Кращий handling для "невідомих" запитів
