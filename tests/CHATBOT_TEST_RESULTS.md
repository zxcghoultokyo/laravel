## 🐛 Виявлені проблеми

### Критичні (P0)

1. **Follow-up запити не працюють**
   - "ще", "ще 5", "покажи більше" - втрачають контекст
   - Файли: `BaseAgent.php` → `extractConversationContext()`, `isFollowupMoreRequest()`

2. **Уточнення фільтрів не працює**
   - "чорного кольору", "дешевше", "дорожче" - не зберігають попередню категорію
   - Файли: `ChatService.php` → `loadSearchState()`, `saveSearchState()`

### Важливі (P1)

3. **Англійські терміни не розпізнаються**
   - "plate carrier" → немає маппінгу на "плитоноска"
   - Файли: `slang_dictionary.php`, `category_aliases` table

4. **Атрибут gender не використовується**
   - "жіноча", "чоловіча" - не фільтрується
   - Файли: `MeiliProductSearchTool.php` → filters

### Середні (P2)

5. **Відповідь "Ось що я знайшов" без продуктів**
   - Коли немає результатів - пусте повідомлення
   - Потрібно: чітке "Не знайшов" або "Спробуйте інший запит"
