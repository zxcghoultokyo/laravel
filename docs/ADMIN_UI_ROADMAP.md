# 🎨 Admin UI/UX Roadmap — Комерційний Продукт

> **Мета:** Перетворити адмінку з dev-інструменту на повноцінний комерційний продукт для магазинів.

## 📊 Поточний стан (Аудит)

### Що є зараз:

| Сторінка | Стан | Проблеми |
|----------|------|----------|
| Dashboard | 🟡 Базовий | Технічні метрики, не бізнес. Немає графіків. |
| Аналітика | 🟡 Функціональний | Перевантажений, немає фільтрів дат, криві графіки |
| Конверсії | 🟡 Базовий | Немає real-time, бракує деталізації |
| Чати | 🟢 Працює | Потребує пошуку/фільтрів |
| Налаштування віджета | 🔴 Критично | Не зберігає в віджет, немає live-preview, бракує 80% функцій |

### Критичні проблеми:

1. **Налаштування віджета не працюють** — колір, текст, позиція НЕ застосовуються до реального віджета
2. **Немає кастомізації бренду** — лого, аватар, шрифти
3. **Немає системи привітань** — не можна налаштувати різні greetings по UTM/категорії
4. **Немає персони/тону** — бот завжди відповідає однаково
5. **Немає промпт-пресетів** — не можна кастомізувати поведінку AI
6. **UI застарілий** — немає dark mode, анімацій, сучасного look & feel
